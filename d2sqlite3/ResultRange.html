    Struct ResultRange

    An input range interface to access the rows resulting from an SQL query.

    The elements of the range are Row structs. A Row is just a view of the current row when iterating the results of a ResultRange. It becomes invalid as soon as ResultRange.popFront() is called (it contains undefined data afterwards). Use QueryCache to store the content of rows past the execution of the statement.

    Instances of this struct are typically returned by Database.execute() or Statement.execute().
