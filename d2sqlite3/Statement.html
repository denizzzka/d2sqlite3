• d2sqlite3

    Built with scod

    Struct Statement

    An SQLite statement execution.

    This struct is a reference-counted wrapper around a sqlite3_stmt* pointer. Instances of this struct are typically returned by Database.prepare().

    Properties

    Name Type Description
    empty [get] bool Tells whether the statement is empty (no SQL statement).
    handle [get] sqlite3.sqlite3_stmt* Gets the SQLite internal handle of the statement.

    Methods

    Name Description
    bind Binds values to parameters of this statement, using parameter index.
    bind Binds values to parameters of this statement, using parameter names.
    bindAll Binds all the arguments at once in order.
    clearBindings Clears the bindings.
    execute Executes the statement and return a (possibly empty) range of results.
    finalize Explicitly finalizes the prepared statement.
    inject Binds arguments, executes and resets the statement, in one call.
    inject Binds the fields of a struct in order, executes and resets the statement, in one call.
    parameterCount Gets the count of bind parameters.
    parameterIndex Gets the index of a bind parameter.
    parameterName Gets the name of the bind parameter at the given index.
    reset Resets a this statement before a new execution.
