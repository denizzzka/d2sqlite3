    Struct Database

    An SQLite database connection.

    This struct is a reference-counted wrapper around a sqlite3* pointer.

    Constructors

    Name Description
    this Opens a database connection.

    Properties

    Name Type Description
    changes [get] int Gets the number of database rows that were changed, inserted or deleted by the most recently executed SQL statement.
    errorCode [get] int Gets the SQLite error code of the last operation.
    handle [get] sqlite3.sqlite3* Gets the SQLite internal handle of the database connection.
    totalChanges [get] int Gets the number of database rows that were changed, inserted or deleted since the database was opened.

    Methods

    Name Description
    attachedFilePath Gets the path associated with an attached database.
    begin Convenience functions equivalent to an SQL statement.
    close Explicitly closes the database connection.
    commit Convenience functions equivalent to an SQL statement.
    createAggregate Creates and registers a new aggregate function in the database.
    createCollation Creates and registers a collation function in the database.
    createFunction Creates and registers a new function in the database.
    enableLoadExtensions Enables or disables loading extensions.
    execute Executes a single SQL statement and returns the results directly.
    isReadOnly Gets the read-only status of an attached database.
    lastInsertRowid Returns the rowid of the last INSERT statement.
    loadExtension Loads an extension.
    prepare Prepares (compiles) a single SQL statement and returngs it, so that it can be bound to values before execution.
    rollback Convenience functions equivalent to an SQL statement.
    run Runs an SQL script that can contain multiple statements.
    setCommitHook Registers a delegate as the database's commit hook. Any previously set hook is released.
    setProfileCallback Sets the profile callback.
    setProgressHandler Sets the progress handler.
    setRollbackHook Registers a delegate as the database's rollback hook.
    setTraceCallback Sets the trace callback.
    setUpdateHook Registers a delegate as the database's update hook.
    tableColumnMetadata Gets metadata for a specific table column of an attached database.
